# 📋 작업 브리프

**원 명령:** 대본검토 진행

## 요약
100개 대본의 실행 단계(Step 1, Step 2) 명확성을 검토하고 최종본을 확정하는 작업을 시작합니다.

## 분배
- **✍️ Writer**: 수집된 100개 대본 전체를 검토하여 각 스크립트의 실행 단계(Step 1, Step 2)가 구체적이고 실행 가능한지 점검하고 최종본을 확정하세요. 특히 실행 단계의 명확성과 논리적 흐름을 강화해야 합니다.
- **📱 Secretary**: 100개 대본 초안의 전체 목록을 Writer에게 전달하고, 검토 진행 상황을 추적할 수 있도록 작업 목록을 정리하세요.
- **✂️ Editor**: Writer가 확정한 스크립트를 기반으로, 각 영상의 시각적 요소(B-roll, 컷 구성)가 명확하게 드러나도록 스크립트 내에 시각적 지침을 추가하는 작업을 준비하세요.
