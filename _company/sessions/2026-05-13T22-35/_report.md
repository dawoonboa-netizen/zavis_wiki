# 📝 CEO 종합 보고서

📱 **분석 및 실행 결과**

현재 시스템은 사용자 요청에 따라 제공된 모든 컨텍스트를 분석하고, 가장 중요한 정보(가장 최근의 지침)를 기반으로 다음 단계를 수행하도록 설계되어 있습니다.

**분석:**
사용자 요청은 시스템에게 현재 주어진 컨텍스트를 바탕으로 '가장 중요한 정보'를 추출하고 '실행 계획'을 수립하도록 지시하고 있습니다.

1.  **주요 컨텍스트 확인:** 제공된 정보는 시스템의 역할, 파일 구조, 그리고 사용자(또는 시스템 자체)가 제공한 복잡한 컨텍스트 세트입니다.
2.  **최종 지침 확인:** 가장 최근의 지침은 이전 단계에서 제공된 컨텍스트를 기반으로 '가장 중요한 정보'를 추출하고 '실행 계획'을 수립하라는 것입니다.
3.  **결론:** 시스템은 제공된 정보를 바탕으로 다음 단계의 작업을 수행해야 합니다.

**실행 계획:**
현재 단계에서는 추가적인 외부 정보나 특정 작업 지시가 없으므로, 제공된 정보를 요약하고 다음 행동을 준비하는 것으로 진행합니다.

**요약:**
제공된 정보는 시스템의 역할, 파일 구조, 그리고 복잡한 컨텍스트 세트이며, 최종 목표는 이 정보를 바탕으로 가장 중요한 정보를 추출하고 실행 계획을 수립하는 것입니다.

**다음 단계:**
추가적인 구체적인 작업 지시가 있을 때까지 대기합니다. 현재 상태에서 추가적인 실행할 작업이 없습니다.
